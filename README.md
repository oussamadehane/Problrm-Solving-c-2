# C++ Problem Solving & Mini Projects

Welcome to my repository! This repository is dedicated to showcasing my solutions for various C++ programming challenges and algorithmic problems, focusing on core programming logic, data structures, and matrix manipulations.

## 🚀 Projects & Problems Included

### Matrix Operations & Analysis
* **Matrix Multiplication Processor:** Generates two 3x3 matrices with random numbers and performs element-wise multiplication, demonstrating efficient memory usage and nested loop logic.
* **Matrix Middle Analyzer:** A specialized utility that extracts and displays the middle row and middle column of a 3x3 matrix using indexed function parameters.
* **Matrix Sum Comparator:** An analytical tool that compares two different 3x3 matrices by calculating the total sum of their elements to check for equality, highlighting boolean logic.
* **3x3 Random Matrix:** Populating a 3x3 matrix with random numbers (1-100) and displaying it in a clean, formatted grid layout.
* **Row Sums to 1D Array:** Extracting the sum of each row from a 3x3 matrix and storing the results in a 1D array.

### Core Challenges & Mini Projects
* **Project 1: Cafe Billing System:** A comprehensive invoicing system using `struct` to manage items, quantities, and final bill calculations including taxes.
* **Project 2: Stone, Paper, Scissors Game:** An advanced implementation using `enum`, `struct`, random numbers, and robust input validation using `cin.fail()`.
* **Challenge: FizzBuzz:** A classic logic puzzle used to test conditional branching and algorithmic thinking.
* **Challenge: Password Strength Checker:** A security validation program that checks password length, character variety, and filters out white spaces.

## 🛠️ Technologies Used
* **Language:** C++
* **IDE:** Visual Studio 2022
* **Key Concepts:** 2D Arrays (Matrices), Enums, Structs, Functions, Input Validation, Memory management (Garbage value prevention).
