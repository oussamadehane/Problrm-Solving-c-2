# C++ Problem Solving & Mini Projects

Welcome to my repository! This repository is dedicated to showcasing my solutions for various C++ programming challenges and algorithmic problems (focused on Course 7 challenges, 2D arrays, matrices, and core programming logic).

## 🚀 Projects & Problems Included

Here is a summary of the key challenges solved in this repository:

* **Problem 01: 3x3 Random Matrix**
    * Populating a 3x3 matrix (2D Array) with random numbers ranging from 1 to 100 and displaying it in a clean grid layout.
* **Problem 03: Row Sums to 1D Array**
    * Extracting the sum of each row from a 3x3 matrix and storing the results in a 1D array before printing.
* **Project 1: Cafe Billing System**
    * A clean invoicing system built using `struct` to manage items, quantity, and final bill calculation including taxes.
* **Project 2: Stone, Paper, Scissors Game**
    * An advanced implementation of the classic game using `enum`, `struct`, random numbers, and robust input validation using `cin.fail()`.
* **Challenge: FizzBuzz**
    * The classic logic puzzle commonly used in technical interviews to test conditional logic.
* **Challenge: Password Strength Checker**
    * A validation program that checks password length, numbers, and filters out white spaces for security.

## 🛠️ Technologies Used

* **Language:** C++
* **IDE:** Visual Studio 2022
* **Concepts applied:** 2D Arrays (Matrices), Enums, Structs, Functions, Input Validation, Memory management (Garbage value prevention).
